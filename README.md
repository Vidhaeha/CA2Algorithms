# CA2Algorithms
This repository contains implementations of various algorithms for educational purposes. The algorithms are categorized into different folders based on their type and functionality.
## Folder Structure
- `SortingAlgorithms/`: Contains implementations of various sorting algorithms such as Quick Sort, Merge Sort, Bubble Sort, etc.
- `SearchAlgorithms/`: Contains implementations of search algorithms like Binary Search, Depth-First Search (DFS), Breadth-First Search (BFS), etc.
- `CA2Algorithms/utils.h`: Contains utility functions and data structures used across different algorithms.
- `CA2Algorithms/BSTNode.h`: Contains the implementation of a Binary Search Tree (BST) node.
- `README.md`: This file, providing an overview of the repository.
